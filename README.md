# LMS(Library Management System) 
## 前端代码

### 1.项目所用技术
一、编程语言
1. HTML
2. CSS
3. JavaScript

二、前端框架
1. Vue.js

三、外部库
1. Font Awesome图标字体库
2. element组件库

### 2.结构
**HTML部分：**

index.html  主页面
register.html 用户注册页面
change_password.html  用户修改密码页面
mainpage.html  主页
categories.html  图书分类页

**CSS部分：**

effects.css  网页特效样式表
main_head.css  网页上边栏样式
其余的为对应同名html的样式表

**JS部分：**

main_head.js  网页上边栏逻辑
effects.js  网页特效逻辑
public.js  页面通用接口

**其他**

fa  本地Font-Awesome图标字体库
lib  本地Vue库以及element组件库
img  网页必要的图片文件
font  网页页面字体（Source HeiTi）
